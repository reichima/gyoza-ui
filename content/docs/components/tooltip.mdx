## 概要

Tooltipコンポーネントは、新しいHTML `popover="hint"` 属性と `interesttarget` 属性を使用した、次世代のツールチップ実装です。JavaScriptによる複雑な制御なしに、ホバーやフォーカスでツールチップを表示できます。

> [!WARNING]
> このコンポーネントは最新のブラウザ機能（`popover="hint"`, `interesttarget`）に依存しています。現在は最新のChrome/Edgeでのみ動作する可能性があります。

## デモ

### 基本的な使い方

export const BasicDemo = () => (
  <div className="my-6 flex gap-4">
    <TooltipTrigger target="tooltip-1">
      ホバーしてください
    </TooltipTrigger>
    <TooltipContent id="tooltip-1">
      <p>これはツールチップです</p>
    </TooltipContent>
  </div>
);

<BasicDemo />

```tsx
import { TooltipTrigger, TooltipContent } from "@/registry/ui/tooltip"

export default function Example() {
  return (
    <div>
      <TooltipTrigger target="tooltip-1">
        ホバーしてください
      </TooltipTrigger>
      <TooltipContent id="tooltip-1">
        <p>これはツールチップです</p>
      </TooltipContent>
    </div>
  )
}
```

## インストール

```bash
npx shadcn@latest add https://your-registry-url.com/r/tooltip.json
```

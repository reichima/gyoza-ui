## 概要

Tooltipコンポーネントは、ホバーやフォーカス時に要素に関連する情報を表示するコンポーネントです。マウスイベントとReact Contextを使用して、シンプルかつ信頼性の高い実装を提供します。

## デモ

### 基本的な使い方

export const BasicDemo = () => (
  <div className="my-6 flex gap-4">

      <TooltipTrigger>

      </TooltipTrigger>

## インストール

```bash
npx shadcn@latest add https://your-registry-url.com/r/tooltip.json
```

## Props

### Tooltip

Tooltipコンポーネントはpropsを受け取りません。子要素としてTooltipTriggerとTooltipContentを含める必要があります。

### TooltipTrigger

`TooltipTrigger`は標準的なHTML div属性をすべてサポートします。

### TooltipContent

## アクセシビリティ

- TooltipContentには`role="tooltip"`が自動的に設定されます
- TooltipTriggerには`aria-describedby`属性が設定され、TooltipContentのIDと関連付けられます
- マウスホバーとキーボードフォーカスの両方で動作します

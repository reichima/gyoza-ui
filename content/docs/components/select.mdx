import { Select, SelectOption } from "@/registry/ui/select";

## 概要

Selectコンポーネントは、新しいCSS機能 `appearance: base-select` を使用して、ネイティブの `<select>` 要素を完全にカスタマイズ可能にします。これにより、JavaScriptによる複雑な再実装なしに、リッチなドロップダウンメニューを作成できます。

> [!WARNING]
> この機能は非常に新しいCSS機能（`appearance: base-select`）に依存しており、現在は実験的なブラウザ機能（Chrome Canaryなど）でのみ動作する可能性があります。

## デモ

### 基本的な使い方

export const BasicDemo = () => (
  <div className="my-6 w-full max-w-xs">
    <Select>
      <SelectOption value="apple">Apple</SelectOption>
      <SelectOption value="banana">Banana</SelectOption>
      <SelectOption value="orange">Orange</SelectOption>
    </Select>
  </div>
);

<BasicDemo />

```tsx
import { Select, SelectOption } from "@/registry/ui/select"

export default function Example() {
  return (
    <Select>
      <SelectOption value="apple">Apple</SelectOption>
      <SelectOption value="banana">Banana</SelectOption>
      <SelectOption value="orange">Orange</SelectOption>
    </Select>
  )
}
```

### カスタムスタイル

`className` プロパティを使用して、セレクトボックスとオプションのスタイルをカスタマイズできます。

export const StyledDemo = () => (
  <div className="my-6 w-full max-w-xs">
    <Select className="bg-blue-50 border-blue-200 text-blue-900">
      <SelectOption value="1" className="text-blue-900">Option 1</SelectOption>
      <SelectOption value="2" className="text-blue-900">Option 2</SelectOption>
      <SelectOption value="3" className="text-blue-900">Option 3</SelectOption>
    </Select>
  </div>
);

<StyledDemo />

```tsx
<Select className="bg-blue-50 border-blue-200 text-blue-900">
  <SelectOption value="1">Option 1</SelectOption>
  <SelectOption value="2">Option 2</SelectOption>
  <SelectOption value="3">Option 3</SelectOption>
</Select>
```

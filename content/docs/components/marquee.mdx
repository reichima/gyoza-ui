## 概要

Marqueeコンポーネントは、水平方向または垂直方向にスクロールするコンテンツを表示するヘッドレスコンポーネントです。アクセシビリティを考慮し、`prefers-reduced-motion`設定に対応しています。

## デモ

    ### ステップ1: コンポーネントファイルを作成

    以下のコードを`components/ui/marquee/marquee.tsx`にコピーしてください。

    ```tsx title="components/ui/marquee/marquee.tsx"
    import * as React from "react"
    import { cn } from "@/lib/utils"

    interface MarqueeProps extends React.ComponentProps<"div"> {
      direction?: "left" | "right" | "up" | "down"
      duration?: number
      pauseOnHover?: boolean
      pauseOnFocus?: boolean
      repeat?: number
      reverse?: boolean
      vertical?: boolean
      fade?: boolean
      respectReducedMotion?: boolean
    }

    function Marquee({
      className,
      direction = "left",
      duration = 30,
      pauseOnHover = false,
      pauseOnFocus = false,
      repeat = 2,
      reverse = false,
      vertical = false,
      fade = false,
      respectReducedMotion = true,
      children,
      style,
      ...props
    }: MarqueeProps) {
      const isVertical = vertical || direction === "up" || direction === "down"
      const isReverse = reverse || direction === "right" || direction === "down"

      return (
        <div
          data-slot="marquee"
          className={cn(
            "group relative flex overflow-hidden",
            isVertical ? "flex-col" : "flex-row",
            fade &&
              (isVertical
                ? "[mask-image:linear-gradient(to_bottom,transparent,white_20%,white_80%,transparent)]"
                : "[mask-image:linear-gradient(to_right,transparent,white_20%,white_80%,transparent)]"),
            className
          )}
          role="region"
          aria-label="スクロールコンテンツ"
          aria-live="off"
          style={
            {
              "--duration": `${duration}s`,
              ...style,
            } as React.CSSProperties
          }
          {...props}
        >
          {Array.from({ length: repeat }).map((_, index) => (
            <div
              key={index}
              className={cn(
                "flex shrink-0",
                isVertical ? "flex-col animate-[marquee-vertical_var(--duration)_linear_infinite]" : "flex-row animate-[marquee_var(--duration)_linear_infinite]",
                isReverse && "[animation-direction:reverse]",
                pauseOnHover && "group-hover:[animation-play-state:paused]",
                pauseOnFocus && "group-focus-within:[animation-play-state:paused]",
                respectReducedMotion && "motion-reduce:animate-none"
              )}
              aria-hidden={index > 0 ? "true" : undefined}
            >
              {children}
            </div>
          ))}
        </div>
      )
    }

    function MarqueeItem({ className, ...props }: React.ComponentProps<"div">) {
      return (
        <div
          data-slot="marquee-item"
          className={cn("shrink-0 px-2", className)}
          {...props}
        />
      )
    }

    export { Marquee, MarqueeItem, type MarqueeProps }
    ```

    ### ステップ2: グローバルCSSにアニメーションを追加

    `app/globals.css`または`styles/globals.css`に以下のキーフレームアニメーションを追加してください：

    ```css title="app/globals.css"
    @keyframes marquee {
      from {
        transform: translateX(0%);
      }
      to {
        transform: translateX(-100%);
      }
    }

    @keyframes marquee-vertical {
      from {
        transform: translateY(0%);
      }
      to {
        transform: translateY(-100%);
      }
    }
    ```
    
    ### ステップ3: インデックスファイルを作成

    エクスポートを簡潔にするため、`components/ui/marquee/index.ts`を作成してください。

    ```ts title="components/ui/marquee/index.ts"

    ```

### MarqueeItem

標準的な`div`要素のpropsを全て受け付けます。

## アクセシビリティ

- `role="region"`と`aria-label`を使用して、スクリーンリーダーにコンテンツの性質を伝えます
- `aria-live="off"`を設定して、スクリーンリーダーが自動的に読み上げないようにしています
- 繰り返されるコンテンツには`aria-hidden="true"`を設定して、スクリーンリーダーが重複して読み上げないようにしています
- `respectReducedMotion`プロップにより、ユーザーのモーション設定を尊重します
- `pauseOnFocus`を有効にすることで、キーボードユーザーがコンテンツを読みやすくします

## カスタマイズ

### スピードの調整

CSS変数`--duration`を使用してスクロール速度を調整できます：

```tsx
<Marquee className="[--duration:10s]">
  {/* 高速スクロール */}
</Marquee>

<Marquee className="[--duration:60s]">
  {/* 低速スクロール */}
</Marquee>
```

### グラデーションのカスタマイズ

`fade`プロップは`mask-image`を使用してグラデーションを実現しています。
カスタムグラデーションを適用する場合は、直接クラスを指定できます：

```tsx
<Marquee className="[mask-image:linear-gradient(to_right,transparent_0%,white_30%,white_70%,transparent_100%)]">
  {/* カスタムフェードエフェクト */}
</Marquee>
```

## 概要

Marqueeコンポーネントは、水平方向または垂直方向にスクロールするコンテンツを表示するヘッドレスコンポーネントです。アクセシビリティを考慮し、`prefers-reduced-motion`設定に対応しています。

## デモ

### 基本的な使い方

<div className="my-6">
  <Marquee className="[--duration:20s]">
    {[
      { name: "山田太郎", username: "@yamada", body: "素晴らしいサービスです！", img: "https://avatar.vercel.sh/yamada" },
      { name: "佐藤花子", username: "@sato", body: "使いやすくて便利です", img: "https://avatar.vercel.sh/sato" },
      { name: "鈴木一郎", username: "@suzuki", body: "デザインが素敵ですね", img: "https://avatar.vercel.sh/suzuki" },
    ].map((review) => (
      <MarqueeItem key={review.username}>
        <Card className="w-[300px]">
          <CardContent className="flex flex-col gap-2 p-4">
            <div className="flex items-center gap-2">
              <img src={review.img} alt={review.name} className="size-8 rounded-full" />
              <div>
                <p className="font-semibold text-sm">{review.name}</p>
                <p className="text-xs text-muted-foreground">{review.username}</p>
              </div>
            </div>
            <p className="text-sm">{review.body}</p>
          </CardContent>
        </Card>
      </MarqueeItem>
    ))}
  </Marquee>
</div>

### フェードエフェクト

<div className="my-6">
  <Marquee fade className="[--duration:15s]">
    {["🎉 新機能リリース!", "⭐ セール開催中", "🚀 パフォーマンス向上"].map((text, i) => (
      <MarqueeItem key={i} className="text-lg font-bold">
        {text}
      </MarqueeItem>
    ))}
  </Marquee>
</div>

### 逆方向スクロール

<div className="my-6">
  <Marquee reverse className="[--duration:18s]">
    {["React", "TypeScript", "Next.js", "Tailwind CSS"].map((tech, i) => (

## インストール

<Tabs items={["CLI", "Manual"]}>
  <Tab value="CLI">
    ```bash
    npx shadcn@latest add marquee
    ```
    
    :::note
    このコンポーネントは、CSSファイル (`marquee.css`) も一緒にインストールされます。
    アニメーション定義がCSS Modulesとして分離されているため、Tailwindに依存せずに使用できます。
    :::
  </Tab>
  <Tab value="Manual">
    ### ステップ1: コンポーネントファイルを作成

    以下のコードを`components/ui/marquee/marquee.tsx`にコピーしてください。

    ```tsx title="components/ui/marquee/marquee.tsx"
    import * as React from "react"
    import { cn } from "@/lib/utils"
    import "./marquee.css"

    interface MarqueeProps extends React.ComponentProps<"div"> {
      direction?: "left" | "right" | "up" | "down"
      duration?: number
      pauseOnHover?: boolean
      pauseOnFocus?: boolean
      repeat?: number
      reverse?: boolean
      vertical?: boolean
      fade?: boolean
      respectReducedMotion?: boolean
    }

    function Marquee({
      className,
      direction = "left",
      duration = 30,
      pauseOnHover = false,
      pauseOnFocus = false,
      repeat = 2,
      reverse = false,
      vertical = false,
      fade = false,
      respectReducedMotion = true,
      children,
      style,
      ...props
    }: MarqueeProps) {
      const isVertical = vertical || direction === "up" || direction === "down"
      const isReverse = reverse || direction === "right" || direction === "down"

      return (
        <div
          data-slot="marquee"
          className={cn(
            "group relative flex overflow-hidden",
            isVertical ? "flex-col" : "flex-row",
            fade &&
              (isVertical
                ? "[mask-image:linear-gradient(to_bottom,transparent,white_20%,white_80%,transparent)]"
                : "[mask-image:linear-gradient(to_right,transparent,white_20%,white_80%,transparent)]"),
            className
          )}
          role="region"
          aria-label="スクロールコンテンツ"
          aria-live="off"
          style={
            {
              "--duration": `${duration}s`,
              ...style,
            } as React.CSSProperties
          }
          {...props}
        >
          {Array.from({ length: repeat }).map((_, index) => (
            <div
              key={index}
              className={cn(
                "flex shrink-0",
                isVertical ? "flex-col" : "flex-row",
                isVertical ? "animate-marquee-vertical" : "animate-marquee",
                isReverse && "[animation-direction:reverse]",
                pauseOnHover && "group-hover:[animation-play-state:paused]",
                pauseOnFocus && "group-focus-within:[animation-play-state:paused]",
                respectReducedMotion && "motion-reduce:animate-none"
              )}
              style={{ animationDuration: "var(--duration)" }}
              aria-hidden={index > 0 ? "true" : undefined}
            >
              {children}
            </div>
          ))}
        </div>
      )
    }

    function MarqueeItem({ className, ...props }: React.ComponentProps<"div">) {
      return (
        <div
          data-slot="marquee-item"
          className={cn("shrink-0 px-2", className)}
          {...props}
        />
      )
    }

    export { Marquee, MarqueeItem, type MarqueeProps }
    ```

    ### ステップ2: CSSファイルを作成

    以下のCSSを`components/ui/marquee/marquee.css`にコピーしてください。

    ```css title="components/ui/marquee/marquee.css"
    @keyframes marquee {
      from {
        transform: translateX(0%);
      }
      to {
        transform: translateX(-100%);
      }
    }

    @keyframes marquee-vertical {
      from {
        transform: translateY(0%);
      }
      to {
        transform: translateY(-100%);
      }
    }

    .animate-marquee {
      animation: marquee linear infinite;
    }

    .animate-marquee-vertical {
      animation: marquee-vertical linear infinite;
    }
    ```
    
    ### ステップ3: インデックスファイルを作成

    エクスポートを簡潔にするため、`components/ui/marquee/index.ts`を作成してください。

    ```ts title="components/ui/marquee/index.ts"
    export { Marquee, MarqueeItem, type MarqueeProps } from "./marquee"
    ```

    フォルダ構造にすることで、関連ファイルがまとまり、管理しやすくなります。

## 使用例

### 基本的な使い方

```tsx
import { Marquee, MarqueeItem } from "@/components/ui/marquee"

export default function Example() {
  return (
    <Marquee>
      <MarqueeItem>アイテム 1</MarqueeItem>
      <MarqueeItem>アイテム 2</MarqueeItem>
      <MarqueeItem>アイテム 3</MarqueeItem>
      <MarqueeItem>アイテム 4</MarqueeItem>
      <MarqueeItem>アイテム 5</MarqueeItem>
    </Marquee>
  )
}
```

### ホバーで一時停止

```tsx
<Marquee pauseOnHover>
  <MarqueeItem>アイテム 1</MarqueeItem>
  <MarqueeItem>アイテム 2</MarqueeItem>
  <MarqueeItem>アイテム 3</MarqueeItem>
</Marquee>
```

### フェードエフェクト

```tsx
<Marquee fade>
  <MarqueeItem>アイテム 1</MarqueeItem>
  <MarqueeItem>アイテム 2</MarqueeItem>
  <MarqueeItem>アイテム 3</MarqueeItem>
</Marquee>
```

### 垂直方向のスクロール

```tsx
<Marquee vertical className="h-[200px]">
  <MarqueeItem>アイテム 1</MarqueeItem>
  <MarqueeItem>アイテム 2</MarqueeItem>
  <MarqueeItem>アイテム 3</MarqueeItem>
</Marquee>
```

### スクロール速度の調整

```tsx
<Marquee duration={20}>
  <MarqueeItem>速いスクロール</MarqueeItem>
  <MarqueeItem>アイテム 2</MarqueeItem>
  <MarqueeItem>アイテム 3</MarqueeItem>
</Marquee>
```

または、CSS変数を使用して調整できます：

```tsx
<Marquee className="[--duration:10s]">
  <MarqueeItem>速いスクロール</MarqueeItem>
  <MarqueeItem>アイテム 2</MarqueeItem>
  <MarqueeItem>アイテム 3</MarqueeItem>
</Marquee>
```

### 逆方向のスクロール

```tsx
<Marquee reverse>
  <MarqueeItem>逆方向</MarqueeItem>
  <MarqueeItem>アイテム 2</MarqueeItem>
  <MarqueeItem>アイテム 3</MarqueeItem>
</Marquee>
```

## Props

### Marquee

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `direction` | `"left" \| "right" \| "up" \| "down"` | `"left"` | スクロール方向 |
| `duration` | `number` | `30` | スクロール速度（秒） |
| `pauseOnHover` | `boolean` | `false` | ホバー時に一時停止するか |
| `pauseOnFocus` | `boolean` | `false` | フォーカス時に一時停止するか |
| `repeat` | `number` | `2` | 繰り返し回数 |
| `reverse` | `boolean` | `false` | 逆方向にスクロールするか |
| `vertical` | `boolean` | `false` | 垂直方向にスクロールするか |
| `fade` | `boolean` | `false` | フェードエフェクトを表示するか |
| `respectReducedMotion` | `boolean` | `true` | `prefers-reduced-motion`設定を尊重するか |

### MarqueeItem

標準的な`div`要素のpropsを全て受け付けます。

## アクセシビリティ

- `role="region"`と`aria-label`を使用して、スクリーンリーダーにコンテンツの性質を伝えます
- `aria-live="off"`を設定して、スクリーンリーダーが自動的に読み上げないようにしています
- 繰り返されるコンテンツには`aria-hidden="true"`を設定して、スクリーンリーダーが重複して読み上げないようにしています
- `respectReducedMotion`プロップにより、ユーザーのモーション設定を尊重します
- `pauseOnFocus`を有効にすることで、キーボードユーザーがコンテンツを読みやすくします

## CSS Modulesでの使用

TailwindクラスをCSS Modulesに置き換える場合は、以下のように実装できます：

```tsx
// marquee.module.css
.marquee {
  position: relative;
  display: flex;
  overflow: hidden;
}

.marqueeVertical {
  flex-direction: column;
}

.marqueeItem {
  flex-shrink: 0;
  padding-left: 0.5rem;
  padding-right: 0.5rem;
}

.marqueeTrack {
  display: flex;
  flex-shrink: 0;
  animation: marquee linear infinite;
}

.marqueeTrackVertical {
  flex-direction: column;
  animation: marquee-vertical linear infinite;
}

.marqueeTrackReverse {
  animation-direction: reverse;
}

.pauseOnHover:hover .marqueeTrack {
  animation-play-state: paused;
}

@keyframes marquee {
  0% {
    transform: translateX(0%);
  }
  100% {
    transform: translateX(-100%);
  }
}

@keyframes marquee-vertical {
  0% {
    transform: translateY(0%);
  }
  100% {
    transform: translateY(-100%);
  }
}

@media (prefers-reduced-motion: reduce) {
  .respectReducedMotion .marqueeTrack {
    animation: none;
  }
}
```

# コンポーネント

## UIコンポーネント

### Accordion

折りたたみ可能なコンテンツセクションを縦に積み重ねたコンポーネントです。

[詳細を見る](/docs/components/accordion)

### Button

汎用的なボタンコンポーネントです。複数のバリアントとサイズをサポートしています。

[詳細を見る](/docs/components/button)

### Card

コンテンツをグループ化するためのカードコンポーネントです。

[詳細を見る](/docs/components/card)

### Combobox

オートコンプリート入力とサジェストリスト付きのコンボボックスコンポーネントです。

[詳細を見る](/docs/components/combobox)

### Dialog

HTML Dialog APIを使用したネイティブなダイアログコンポーネントです。

[詳細を見る](/docs/components/dialog)

### Input

テキスト入力フィールドコンポーネントです。

[詳細を見る](/docs/components/input)

### Label

フォーム要素のラベルコンポーネントです。

[詳細を見る](/docs/components/label)

### Popover

HTML Popover APIを使用したネイティブなポップオーバーコンポーネントです。

[詳細を見る](/docs/components/popover)

### Select

CSSでカスタマイズ可能なネイティブセレクトコンポーネントです。

[詳細を見る](/docs/components/select)

### Textarea

複数行のテキスト入力コンポーネントです。

[詳細を見る](/docs/components/textarea)

### Tooltip

ホバーやフォーカス時に情報を表示するツールチップコンポーネントです。

[詳細を見る](/docs/components/tooltip)

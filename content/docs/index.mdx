# Gyoza UIコンポーネントライブラリ

Gyoza UIは、ユーザビリティとカスタマイズのしやすさを強く意識した実験的なコンポーネントライブラリです。

## クイックスタート

shadcn CLIを使用してコンポーネントをインストールできます：

```bash
npx shadcn@latest add https://your-registry-url.com/r/button.json
```

## コンポーネント一覧

このレジストリには以下のコンポーネントが含まれています：

- Button - ボタンコンポーネント
- Card - カードコンポーネント
- Input - 入力フィールド
- Label - ラベル
- Textarea - テキストエリア

詳細は各コンポーネントのドキュメントをご覧ください。

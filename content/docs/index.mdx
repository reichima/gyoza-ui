# Gyoza UIコンポーネントライブラリ

Gyoza UIは、shadcn/uiをベースにしたカスタムコンポーネントレジストリです。再利用可能なUIコンポーネントを提供し、あなたのプロジェクトに簡単に統合できます。

## 特徴

- 🎨 **美しいデザイン** - Tailwind CSSで構築されたモダンなコンポーネント
- 🔧 **カスタマイズ可能** - ソースコードを直接編集可能
- 📦 **簡単なインストール** - shadcn CLIで即座にインストール
- ⚡ **高速** - Next.js 15とReact 19をサポート

## クイックスタート

shadcn CLIを使用してコンポーネントをインストールできます：

```bash
npx shadcn@latest add https://your-registry-url.com/r/button.json
```

## コンポーネント一覧

このレジストリには以下のコンポーネントが含まれています：

- Button - ボタンコンポーネント
- Card - カードコンポーネント
- Input - 入力フィールド
- Label - ラベル
- Textarea - テキストエリア

詳細は各コンポーネントのドキュメントをご覧ください。
